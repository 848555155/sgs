To build:

1. Download and install Blend for WPF SDK. (Around 2.6M)
   http://www.microsoft.com/en-us/download/details.aspx?id=10801

2. Open Sanguosha.sln with VS 2012 under Source/Sanguosha.

3. Set UI/Sanguosha as startup project in VS 2012.

* If VS reports "Build succeeded" or if "Sanguosha.exe" is produced in your Bin/Debug directory, ignore any compiler errors.

** The following packages are required if you are using VS 2010:
   a. Visual Studio 2010 SP1.
      http://www.microsoft.com/en-us/download/details.aspx?id=23691
   b. SQL Server 2012 Express LocalDB. (Around 30MB)
      http://www.microsoft.com/en-us/download/details.aspx?id=29062
   c. SQL server 2012 data-tier application framework. (Around 3MB)
      http://www.microsoft.com/en-us/download/details.aspx?id=35756

To run:

Go to Bin/Debug, run Sanguosha.exe.
